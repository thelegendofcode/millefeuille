# User Personas

Middle aged guy saving for retirement. Mainly adopts a buy and hold strategy but does some selling once in a while. Asset allocation is indexes and tech stocks, no bonds. Has both taxed and tax deferred accounts. Income in Yen while portfolio is mostly US$. 

Wants to use app to make easier purchase decisions based on target *Asset Allocation* and current portfolio.

Wants to see growth of portfolio over time.

Wants to see which stock has made the most growth over time, to be able to create a tax efficient selling plan.

Wants to be able to mark mile stones.

Wants to be able to see predicted future value of portfolio.

Wants to see history of all trades made.
